ngOnChange Hook

Angular provides lifecycle hooks for change detection.

OnChanges is an interface and has a method declaration i.e ngOnChanges()

In this example : parent component changes the value of child component Using @Input decorator

When value changes in the child component , ngOnChange methord runs automatically

The method ngOnChanges() uses SimpleChanges as an argument that gives new and previous value of input values after changes

SimpleChange Class and SimpleChanges Interface

SimpleChanges Use following property to detect changes -:

previousValue : Keeps previous property of input property

currentValue: Keeps current value of input property.

isFirstChange() : Boolean value that tells whether the new value is the first value assigned.
